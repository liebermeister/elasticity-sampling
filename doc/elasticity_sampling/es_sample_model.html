es_sample_model

PURPOSE

ES_SAMPLE_MODEL - Sample all model parameters for a network

SYNOPSIS

function [result, es_options, es_constraints] = es_sample_model(N, W, ind_ext, es_constraints, es_options)

DESCRIPTION 

 ES_SAMPLE_MODEL - Sample all model parameters for a network

 result = es_sample_model(N, W, ind_ext, es_constraints, es_options)
 
 Sample all numbers needed for a kinetic in steady state

 This script returns one model instance, including the sampled elasticities 

 Sampling happens in two stages: 
  1. Steady state         (es_sample_steady_state_state.m)
  2. Elasticities and MCA (es_sample_elasticities.m)

 For the information contained in 'es_constraints' and 'es_options', see es_default_es_options.m
